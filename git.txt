repository:仓库（版本库）


一、linux Command

ls -a   查看目录下的文件

mkdir   在工作目录下建立一个子目录

cd ..   返回上一级目录

pwd     显示当前目录绝对路径



二、git Command

git init初始化git仓库

git status看文件状态，红色Untracked files是不被追踪的

git diff   查看具体修改的内容

git add添加文件追踪（将文件从工作区添加到暂存区）

git commit -m 'add style.css'（将文件从暂存区添加到版本库）

git add .一次性将所有文件添加到追踪

git commit -m '本次提交说明'一次性将所有新文件添加到版本库

git log查看提交日志

git commit -m'modify index.html'更新

git commit -am 'remodified index.html'一次性提高所有修改文件到版本库

git rm    删除文件

git push -u origin master 当commit完后就可以push推送更新了，第一次推送改动可以使用 -u 参数，使用之后会绑定你这一次的仓库分支名，这样的话下一次推送就不需要加上分支名了
